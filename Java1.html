Java Assignment

1. Literals (Fixed Values in Code)

Definition: Literals are fixed values assigned directly to variables in the program. They are hard-coded constants and are interpreted literally by the Java compiler.

Explanation: They represent values like numbers, characters, text, and logical values in your Java code. Whenever you assign a value like 10, 'A', or "Java" to a variable, you are using a literal.

Example:

int x = 100; // Integer literal

2. Keywords

Definition: Keywords are predefined, reserved words in Java. Each keyword has a specific purpose and cannot be used as names for variables, methods, or classes.

Explanation: They help Java understand what operation or behavior you are trying to implement, like declaring variables, loops, methods, classes, etc.

Examples:

class, interface       → Declaring classes and interfaces
if, else, switch       → Control flow
try, catch, finally    → Exception handling
static, void, new      → Object and method handling

3. Identifiers

Definition: Identifiers are names used to identify classes, methods, variables, objects, etc.

Explanation: They help label and reference the components in your program. You choose them to make your code readable and meaningful.

Rules:

  • Can't be Java keywords
  • Can start with a letter, $, or _
  • Case-sensitive

Examples:

Student, employeeId      → Valid identifiers
_temp, $data             → Valid but not commonly used
int, class               → Invalid (keywords)

4. Arrays

Definition: An array is a collection of elements of the same data type, stored at contiguous memory locations.

Explanation: Arrays allow you to store multiple values of the same type in a single variable, instead of declaring separate variables.

int[] numbers = {1, 2, 3, 4, 5};

5. Methods (All Scenarios)

Definition: A method is a block of code that performs a specific task. Methods improve code reusability and organization.

Explanation: Java methods can be static or non-static, with or without parameters, and may return a value or not. You can also overload methods.

Type          Example                            Description
Normal        void greet()                       Regular method
Static        static int add(int a, int b)       Belongs to class
Overloaded    void show(int); void show(String); Same name, diff params
Varargs       void display(String... names)      Accepts variable arguments
Recursion     int fact(int n) { return n==0 ? 1 : n * fact(n-1); }

6. Operators

Definition: Operators are symbols used to perform operations like addition, comparison, assignment, and more on variables and values.

Explanation: Operators allow you to write logic and expressions in your program.

Category     Examples         Use
Arithmetic   +, -, *, /, %     Math operations
Relational   ==, !=, >, <      Compare values
Logical      &&, ||, !         Logical conditions
Bitwise      &, |, ^, ~        Binary-level operations
Shift        <<, >>, >>>       Bit shifting
Assignment   =, +=, -=         Assigning values
Ternary      a > b ? a : b     Short if-else
Instanceof   obj instanceof String   Check object type

7. Looping Techniques

Definition: Loops are control structures that repeat a block of code as long as a condition is true.

Explanation: They are used for repetition, especially when the number of iterations is known or condition-based.

Type            Syntax                             Description
for             for (int i = 0; i < n; i++)        Fixed iterations
while           while(condition)                   Unknown iterations
do-while        do { ... } while(condition);       Runs at least once
Enhanced for    for (int x : arr)                  Arrays/collections
Nested          for (...) { for (...) { ... } }    Loop inside a loop
