1. Literals

Literals are fixed values assigned to variables. They represent constant values in Java programs.

  • Integer Literal: int age = 25;
  • Floating Point Literal: float pi = 3.14f;
  • Character Literal: char grade = 'A';
  • String Literal: String name = "Shivani";
  • Boolean Literal: boolean isValid = true;

2. Keywords

Keywords are reserved words in Java that have predefined meanings.

Examples: class, if, else, return, void, static, etc.

You cannot use keywords as identifiers (variable or method names).

3. Identifiers

Identifiers are the names used for classes, variables, methods, etc.

  • Must start with a letter, underscore (_), or dollar sign ($).
  • Cannot start with digits or use keywords.
  • Are case-sensitive.
class Student {
    int rollNumber; // Identifier
    void displayName() {
        // Method identifier
    }
}

4. Arrays

Arrays are used to store multiple values of the same type in a single variable.

int[] numbers = {1, 2, 3, 4, 5};
String[] names = new String[3];
names[0] = "Alice";
names[1] = "Bob";
names[2] = "Charlie";

5. Methods

Methods define a block of code that can be executed when called. Methods support modular and reusable code.

Types of Methods:

  • Void Method (No return):
    • public static void greet() {
    System.out.println("Hello!");
}
  • Return Method:
    • public static int add(int a, int b) {
    return a + b;
}
  • Method Overloading:
    • public static void show(String name) {
    System.out.println(name);
}

public static void show(int age) {
    System.out.println(age);
}
  • Static vs Non-Static Methods:
    • public static void staticMethod() {
    System.out.println("Static method");
}

public void instanceMethod() {
    System.out.println("Instance method");
}

6. Operators

Java provides various operators to perform operations on variables and values.

  • Arithmetic: +, -, *, /, %
  • Relational: ==, !=, >, <, >=, <=
  • Logical: &&, ||, !
  • Assignment: =, +=, -=, *=, /=
  • Unary: ++, --, +, -
  • Bitwise: &, |, ^, ~, <<, >>

7. Looping Techniques

Loops are used to execute a block of code multiple times.

Types of Loops:

  • For Loop:
    • for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
  • While Loop:
    • int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}
  • Do-While Loop:
    • int i = 0;
do {
    System.out.println(i);
    i++;
} while (i < 5);
  • For-Each Loop (for arrays):
    • String[] colors = {"Red", "Green", "Blue"};
for (String color : colors) {
    System.out.println(color);
}
